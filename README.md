# Longmont Vape & Smoke Website

Static website ready for GitHub Pages.

## Upload to GitHub

1. Open your `longmont-vape` repository.
2. Choose **Add file → Upload files**.
3. Upload `index.html`, `styles.css`, `script.js`, and `favicon.svg`.
4. Commit the changes.
5. Go to **Settings → Pages**.
6. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
7. Save.

Your website address will be similar to:

`https://YOUR-USERNAME.github.io/longmont-vape/`
